## About the host — Moss (YC F25)

Moss is a real-time semantic-search runtime for conversational AI / AI agents. It runs search locally inside the agent runtime for **sub-10ms lookups with zero infrastructure**, built in **Rust and WebAssembly** — avoiding the 100–500ms latency of remote database calls during multi-turn conversations. It powers voice agents, copilots, and multimodal apps. Built by InferEdge Inc. (San Francisco, YC-backed); lists Microsoft, EPAM, UC Berkeley, Carnegie Mellon, Podium, and Stanford among users. ([moss.dev](https://www.moss.dev/), [Moss on YC](https://www.ycombinator.com/companies/moss), [Launch YC: Moss](https://www.ycombinator.com/launches/Oiq-moss-real-time-semantic-search-for-conversational-ai))
